 ## 프로젝트 소개
포인트를 활용한 음원 추천 커뮤니티 및 차트 서비스

### 팀원 소개
- 기획 : 석시윤
- PM : 전승환
- Infra : 황준하
- Front : 권소연
- CTO : 김준호

## 프로젝트 기술스택

- 자바 : JAVA 17
- 개발 툴 : SpringBoot 3.0.1
- 필수 라이브러리 : SpringBoot Web, MySQL, Spring Data JPA, Lombok, Spring Security, JWT, Spring Validation, QueryDsl, OAuth2, Thymeleaf
- 빌드 : Gradle 7.4
- DB : MySql
- CLOUD : AWS EC2, AWS RDS, AWS S3
- CI/CD : Docker, GitLab
- API: Spority API
